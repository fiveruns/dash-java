FiveRuns Dash Java API

The Dash Java API provides basic metric support for Java applications. It does not support traces or exceptions as of this release.

Application Integration

To use Dash in your application, you need to:

  • Add Java code to your application which configures and starts Dash.
  • Add Dash and its dependencies to your application's classpath.

Configuring Dash

Dash is configured via a simple Java API. Import the core Dash classes: 

  import com.fiveruns.dash.*;
Create a list of metric recipes that you wish to monitor: 
  Recipe[] recipes = {
      Recipe.getJvmRecipe(),
  };
Pass in your application token and recipe listing to the Plugin.start method. 
    Plugin.start("change-me-to-your-app-token", recipes);
This will spin off a thread which will regularly collect and send metrics to Dash.

Running Dash

Dash has three runtime dependencies. You will need to add the Dash jar along with Apache's commons-httpclient 3.1, commons-logging 1.0.4 and commons-codec 1.2 to your classpath.
